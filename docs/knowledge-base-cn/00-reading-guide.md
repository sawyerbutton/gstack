# gstack 深度知识库：阅读路径指南

> gstack 是 Y Combinator CEO Garry Tan 开源的 AI 软件工厂。它的价值不在于工具本身，而在于它把世界顶级创业导师和工程管理者的思维方式编码成了可执行的 AI 行为。本知识库是对这套思维方式的深度拆解。

---

## 这个知识库是什么

gstack 仓库里有 22 个 SKILL.md 模板文件，总计超过 5000 行 prompt。这些 prompt 不是简单的指令——它们编码了角色认同、认知模式、强制约束和结构化流程。

本知识库将这些散布在代码中的方法论提炼成 12 篇独立文章，按**思维方式、系统设计、实战拆解**三个维度组织。每篇文章都可以独立阅读，但它们之间存在逻辑递进关系。

## 关于作者背景

Garry Tan — Y Combinator 现任 CEO/总裁。曾与 Coinbase、Instacart、Rippling 等数十亿美元公司的创始人在车库阶段合作。此前是 Palantir 早期工程经理/PM/设计师，联合创办了 Posterous（后卖给 Twitter），2013 年构建了 YC 的内部社交网络 Bookface。

他在 2026 年前三个月的 GitHub 贡献超过 1237 次，日均 1-2 万行可用代码。gstack 是他实现这个产出的方法论。
